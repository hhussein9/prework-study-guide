# Prework Study Guide Webpage

## Description

This Prework Study Guide was created for boot camp students who were going through the Prework. It contains notes on HTML, CSS, Git, and JavaScript.

## Installation

N/A

## Usage

To use the Prework Study Guide, one can begin by reviewing the notes in each section. For guidance on which topic to study first, the user should open the Chrome DevTools by pressing Command+Option+I (macOS) or Control+Shift+I (Windows). This action will result in a console panel opening either below or to the side of the webpage in the browser. Within this console panel, users will find a list of the prework topics covered, along with a recommendation on which topic to study first.
